# Roadmap

This roadmap is focused on adoption quality first, then feature expansion.

## Week 1

- Improve onboarding conversion in README and setup flow.
- Tighten contributor path for first external PRs.
- Capture baseline growth metrics.

## Week 2

- Add issue templates and PR template.
- Publish reproducible demo prompts and output examples.
- Improve triage speed and label hygiene.

## Week 3-4

- Add benchmark examples comparing uncited vs cited Unity answers.
- Expand MCP integration guides for additional clients.
- Prioritize top friction points from new-user feedback.
