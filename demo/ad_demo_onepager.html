Q2 - MeshRenderer receiving 2D Lights in URP 2D Renderer

UnityRAG: faster + correct

With UnityRAG MCP

In Unity, how can I make MeshRenderer objects receive 2D Lights in URP's 2D Renderer?

[thought for 48 seconds...]

As of Unity 6.3 (6000.3), `MeshRenderer` objects can receive URP 2D Lights if you use a 2D-compatible material/shader. Use URP 2D Renderer + Mesh2D-Lit-Default, or a Sprite Lit/Custom Lit Shader Graph with Sort 3D As 2D Compatible.
